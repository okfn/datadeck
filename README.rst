DataDeck:
=========

Integrating ckanjs library onto a pure desktop app

Further information:
* ckanjs running: http://dev.okfn.org/ckanjs/app/
* ckanjs source code: http://bitbucket.org/okfn/ckanjs

Dependencies:
============
* Python >= 2.5
* python-gconf
* python-gobject
* python-webkit
